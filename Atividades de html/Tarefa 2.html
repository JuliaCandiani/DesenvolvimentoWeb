Sobre W3C

Acessibilidade

    O padrão W3C (World Wide Web Consortium) trata-se de uma grande organização internacional com mais de 400 membros, entre empresas, órgãos governamentais e associações independentes, responsável pelos protocolos e padronização da World Wide Web, a rede mundial de computadores. Ou seja: a missão dessa entidade é desenvolver e criar protocolos para interpretação de todo o conteúdo da internet. De uma forma geral, todo site criado por meio desse padrão estará disponível, visível e com potencial máximo para acesso das pessoas em qualquer plataforma independente de suas características. O responsável pela criação do W3C é Tim Berners-Lee.
    A seguir pode-se observar alguns desse padrões e suas aplicações:
  • Web Design e Aplicações (HTML5, CSS, SVG e AJAX)
  • Arquitetura Web (HTTP e URI)
  • Web Semântica (RDF, SPARQL, OWL e SKOS)
  • Tenologia XML (XML, XQUERY, XML SCHEMA e XSLT)
  • Web Services (HTTP, XML, SOUP, WSDL e SPARQL)
  • Web de Dispositivos
  • Padrões Web
